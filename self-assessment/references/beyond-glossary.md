# Beyond Glossary

## Industry Acronyms

| Term | Definition |
|------|-----------|
| ADR | Average Daily Rate (Total Revenue per booking / night short-term booked) |
| ALOS | Average Length of Stay |
| CM | Channel Manager |
| LOS | Length of Stay. Duration of a reservation. Often associated with LOS Discounting/Pricing |
| LOS Discount / Pricing | Discounts/prices based on the number of days in a reservation |
| MLOS | Minimum Length of Stay |
| OTA | Online Travel Agencies |
| PM | Property Manager or Property Management |
| PMS | Property Management System |
| PMC | Property Management Company. A professional vacation rental management company (e.g. Vacation Company, Southern Resorts) |
| PO | Paid Occupancy |
| POAN | Paid Occupancy (Available Nights) |
| POB | Pay on Booking |
| POS | Pay on Stay |
| RevPar | Revenue per Available Unit |
| STR | Short Term Rental |
| VRMA | Vacation Rental Management Association |

## Beyond-Specific Acronyms

| Term | Definition |
|------|-----------|
| ABP | Automated Base Price |
| ABR | Automatic Booking Review — sends recommendations automatically |
| BART | Booking Analysis Recommendations Tool |
| CCR | Cancellable Customer Billable — billable earned but check-in hasn't occurred |
| CRP | Customer Revenue Processed |
| COMPS | Comparison Prices — pricing other competitors are using |
| COMP SETS | A set of listings you compare a listing to (Competitive Set) |
| Contracted Account | A customer signed by the Sales team, given SMB (Lighthouse Overview) or traditional CSM service levels |
| Dedicated Account | A Contracted account with a CSM assigned |
| DCMs | Dynamic Channel Markups (a feature of Relay) |
| EBR | Executive Business Reviews |
| IKT | Internal Knowledge Transfer |
| LAM | Listed in Active Market — prospect's or customer's PMS listings that are listed and geographically located in a live market that Beyond can price |
| LAMEN | Listed in Active Market and Enabled — listings in LAM for which price posting in Beyond is enabled |
| Lead Time | Time between when the reservation is made and check-in (also called Booking Window) |
| LISTINGS (in Admin) | All listings available in connected PMS |
| LISTED (L) (in Admin) | All valid PMS listings |
| LMD | Last Minute Discounts and Future Increases (now called Time Based Adjustments) |
| MaxEF | Maximum Event Factor |
| MeF | Minimum Event Factor |
| Monthly Credit | A contracted and automated recurring fixed monthly credit that does not roll over (e.g. $200/month). Alternative form of discounting to dropping the percentage rate |
| MLSPP | Market Level Search Powered Pricing |
| Net Cash Collected | Money collected via credit card or paid invoice less cash refunds |
| RAD | Request a Demo |
| RevPan | Revenue per Available Night |
| Self-Service / Uncontracted Account | A customer without a contract who signs up via the website. Tend to be smaller annual value deals |
| SeSe | Self-Service |
| SHOWW | See How Our Week Went |
| SPP | Search Powered Pricing |
| Statement | Appears in product — what the customer sees and can audit statement items on |
| Statement Items | Revenue, cancellations, or adjustments that appear on the statement |
| STL | Seasonal Trend Decomposition using LOESS (STL Decomposition) |
| Strops | Strategy & Business Operations |
